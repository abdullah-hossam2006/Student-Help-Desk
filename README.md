# Student Help Desk System

## Project Idea
A client-server JavaFX application that allows students to create support tickets and admins to manage and reply to them using TCP socket networking.

## Features
- Login and Register
- Student Dashboard
- Create Tickets
- View My Tickets
- Admin Dashboard
- Reply to Tickets
- Real TCP Socket Communication
- Multiple Clients Support

## Technologies Used
- Java
- JavaFX
- TCP Sockets
- IntelliJ IDEA

## How to Run

### Run Server
